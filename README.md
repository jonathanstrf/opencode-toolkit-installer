# OpenCode Toolkit Installer

Unified installer for Spartan AI Toolkit + Superpowers on OpenCode.

Quick start

1. Clone locally:
   git clone <your-repo-url> ~/Code/opencode-toolkit-installer
2. Run installer (dry run):
   bash scripts/setup-opencode-toolkits.sh --dry-run
3. Real run:
   bash scripts/setup-opencode-toolkits.sh

See scripts/ for the installers and tests/
